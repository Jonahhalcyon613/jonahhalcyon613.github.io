aftr brings the power of Puppeteer to Adobe After Effects. It allows you to control your video projects using code. Use this tool alongside Claude Code to build professional videos without manual labor. This tool bridges the gap between text-based commands and complex visual effects.

🚀 Getting Started

Follow these steps to set up aftr on your Windows computer. You do not need to be a developer to use this tool, but you do need Adobe After Effects installed on your machine.

⚙️ System Requirements

  • Windows 10 or 11.
  • Adobe After Effects (installed and licensed).
  • Node.js (latest long-term support version).
  • A stable internet connection for the automation process.

🛠️ How it Works

The software acts as a bridge between your computer commands and the After Effects engine. When you run a task, aftr translates your request into instructions that After Effects understands. This allows for rapid rendering and layer manipulation that would take hours to perform manually.

💡 Common Use Cases

Users leverage aftr for repetitive motion graphics tasks. If you create template-based videos with changing text or images, this tool handles the heavy lifting. It connects with FFmpeg to handle video files quickly and efficiently.

⚠️ Troubleshooting

If the program fails to launch, verify that After Effects is not currently open. Sometimes, multiple instances of the software can cause errors. Ensure you have the latest version of After Effects installed to maintain compatibility with the plugin.
